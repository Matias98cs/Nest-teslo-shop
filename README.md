# Teslo API

1. Clonar Proyecto
2. ```yarn install`
3. Clonar el archivo ```.env.template``` y renombrar a ```.env```
4. Cambiar a las variables de entorno

5. Levantar la DB

```
docker-compose up -d
```

6. Levantar: ```yarn:start:dev```
